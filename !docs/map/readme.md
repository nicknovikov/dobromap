# Карты для отрисовки и настройки

Карты в формате SVG предназначены для предварительной работы с контурами в векторном редакторе.
Сформированные контуры экспортируются в формат SVG и копируются в файл `src/components/regions.json`
В качестве источника контуров использовался https://mapsvg.com/maps

## Оригиналы карт

Оригиналы карт являются исходными данными и не предназначены для редактирования.

- `russia.orig.svg` - оригинал карты России
- `belarus.orig.svg` - оригинал карты Белоруссии
- `kazakhstan.orig.svg` - оригинал карты Казахстана
- `ukraine.orig.svg` - оригинал карты Украины
- `georgia.orig.svg` - оригинал карты Грузии
- `armenia.orig.svg` - оригинал карты Армении

## Рабочая карта

Рабочая карта предназначена для редактирования данных.

- `map.ai` - рабочая карта для редактирования в Illustrator и экспорта в `map.svg`
- `map.svg` - рабочая карта, полученная после экспорта из `map.ai` для копирования данных SVG в `regions.json`
